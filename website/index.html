FAQ

常见问题

下载后打不开怎么办？

先检查系统安全策略和签名拦截，必要时以管理员身份启动一次。

出现 network error 怎么办？

请优先确认本地 API 是否在 `127.0.0.1:2026` 正常运行。

能离线使用吗？

工具和文件能力可离线执行，但模型推理仍需要模型端点可访问。
