INSTALL GUIDE

安装说明

1. 下载对应系统包

优先使用“最新版本”资产，避免手动选择错误架构。

2. 校验文件哈希

点击“复制校验码”，在本地对照 SHA256，确认安装包完整。

3. 首次启动初始化

启动后配置模型与 API，即可进入任务执行与工具模式。

FAQ

常见问题

为什么下载后点击无反应？

请确认系统安全策略没有阻止应用启动，并优先使用管理员身份运行一次。

出现 network error 怎么办？

多数情况是本地 API 进程未运行或端口不一致，请检查 2026 端口并重启应用。

可以离线使用吗？

工具与文件能力可离线执行，但模型推理仍需可访问的模型服务端点。
