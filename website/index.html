FAQ

常见问题

下载后无响应怎么办？

请先检查系统安全策略与签名拦截，再以管理员权限启动一次。

出现 network error 怎么处理？

多数场景是本地 API 未启动或端口不一致，请优先检查 `127.0.0.1:2026`。

支持离线吗？

文件与工具操作可本地执行，但模型推理依赖可访问的模型服务端点。
